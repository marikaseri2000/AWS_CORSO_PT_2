Seconda Parte del Percorso – Progetti Applicativi e Strutturazione del Codice

Questa repository contiene esercitazioni, progetti e materiale didattico sviluppati durante la seconda parte del percorso formativo AWS re/Start, organizzato da Edgemony in collaborazione con Amazon Web Services (AWS).

In questa fase del corso l’attenzione è posta sull’applicazione pratica delle competenze acquisite, con particolare focus su programmazione in Python, strutturazione del codice, logica applicativa e gestione di piccoli progetti software.

Ambito Tecnico

I contenuti presenti nella repository affrontano i seguenti aspetti:

sviluppo di applicazioni Python strutturate

utilizzo di classi e moduli

gestione dello stato dell’applicazione

organizzazione del codice in più file

problem solving e logica algoritmica

simulazione di casi d’uso reali

Struttura della Repository

La repository è organizzata in cartelle tematiche e lezioni per data:

Cartelle per data ( 08-01-2026, 12-01-2026)
Contengono il codice sviluppato durante le singole lezioni, gli esempi svolti e gli esercizi di approfondimento;

sfida

Obiettivi della Seconda Parte

La seconda parte del percorso AWS re/Start mira a:

sviluppare autonomia nella progettazione di applicazioni

applicare concetti di programmazione orientata agli oggetti

migliorare la capacità di organizzare il codice in modo modulare

simulare contesti applicativi simili a quelli aziendali
